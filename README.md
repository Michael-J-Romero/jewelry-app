# Jewelry App

A polished Next.js starter for a jewelry storefront using the App Router, TypeScript, Material UI with Emotion, Motion, and Embla Carousel.

## Stack

- Next.js
- App Router
- TypeScript
- Material UI + Emotion
- Motion
- Embla Carousel
- Turbopack for local development

## Included packages

- @mui/material
- @emotion/react
- @emotion/styled
- motion
- embla-carousel-react

## Getting started

1. Install dependencies with npm install
2. Start the development server with npm run dev
3. Open the local URL shown in the terminal

## Starter structure

- app for the App Router pages and layout
- components for the hero section, carousel, and theme registry
- theme for the Material UI theme
