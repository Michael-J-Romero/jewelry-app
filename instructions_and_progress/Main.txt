1. PRODUCT DEFINITION

This page is a hybrid between a luxury ecommerce interface and an interactive styling tool.

It must feel like:

a natural extension of NorVoch’s storefront
not a separate “app”
not overly experimental
Core user mindset

The user is:

browsing jewelry
discovering pieces
visualizing combinations
building a curated ear stack
intending (or considering) purchasing
Core experience loop
User sees ear + catalog
User selects a piercing
Catalog updates to relevant products
User browses + filters
User places jewelry
Visual updates instantly
Cart awareness increases
User continues building or purchases

The NorVoch Jewelry Builder should feel like a refined extension of the brand’s ecommerce experience—balancing elegance, clarity, and product richness.
